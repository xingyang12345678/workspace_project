# AI 数据工作台（Workspace）使用文档

本项目是一个本地运行的 **AI 数据工作台**：管理数据集、可视化 JSONL、执行 `tools/` 下的 Python 脚本、按步骤执行 Pipeline、快速记录个人知识，并预留插件扩展结构。

## 1. 目录结构与概念

### 1.1 项目目录（代码）

- `backend/`：FastAPI 后端
- `frontend/`：React + Vite 前端
- `docs/`：项目文档
- `workspace_project/`：默认 workspace（运行时数据目录）

### 1.2 Workspace 目录（数据）

Workspace 是应用围绕的“本地工作区根目录”，包含：

- `datas/`：数据集（支持子目录分类，常见 JSONL）
- `tools/`：Python 工具脚本（一个脚本一个小功能）
- `pipelines/`：Pipeline 定义（JSON）
- `backup/`：备份（只增不减，hash 去重）
- `knowledge/`：知识记录
- `plugins/`：插件预留目录（MVP 不实现插件系统）

默认 workspace：`/home/xxy/ai-data-workspace/workspace_project/`

你也可以用环境变量指定 workspace：

```bash
export WORKSPACE_ROOT="/abs/path/to/your_workspace"
```

## 2. 环境准备

- Python：建议 3.10+
- Node.js：建议 18+

## 3. 启动后端（FastAPI）

```bash
cd /home/xxy/ai-data-workspace/backend
pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

后端接口文档：
- OpenAPI/Swagger：`http://127.0.0.1:8000/docs`

## 4. 启动前端（React）

```bash
cd /home/xxy/ai-data-workspace/frontend
npm install
npm run dev
```

访问：`http://localhost:5173`

说明：前端已配置 Vite 代理，会把 `/api/*` 转发到 `http://127.0.0.1:8000`。

## 5. 功能使用说明（MVP）

### 5.1 文件导航（左侧）

- 点击目录：展开/收起（支持打开与关闭）
- 点击 `datas/` 下的 `.jsonl`：进入数据集浏览页
- 点击 `tools/`：进入工具执行页
- 点击 `pipelines/`：进入 Pipeline 页
- “备份 datas”：一键备份 `datas/` 到 `backup/`（hash 去重，只增不减）

顶部按钮：
- “打开/关闭导航”：隐藏或显示左侧文件导航
- “打开/关闭终端”：隐藏或显示底部终端面板

### 5.2 JSONL 数据可视化（DatasetViewer）

入口：在左侧 `datas/` 目录中点击 `.jsonl` 文件。

支持：
- 上一条 / 下一条
- 跳转 index（0-based）
- 优先按聊天字段展示：
  - `messages`
  - `chosen`
  - `rejected`
- 若不包含上述字段，则显示整条 JSON 的格式化内容

示例数据：`workspace_project/datas/sample.jsonl`

### 5.3 Tool 执行（ToolRunner）

工具来源：workspace 的 `tools/` 目录下 `*.py` 文件（自动扫描）

执行方式：
- 在 UI 中选择工具
- 输入参数（空格分隔）
- 点击执行
- 输出（stdout/stderr/exit code）显示在底部“终端面板”

示例工具：`workspace_project/tools/echo_sample.py`

### 5.4 终端面板（TerminalPanel）

用于展示：
- 工具 stdout/stderr
- pipeline 步骤执行输出
- 错误信息

支持：
- 清空
- 顶部按钮打开/关闭

### 5.5 Pipeline（基础版）

Pipeline 定义存储在 workspace 的 `pipelines/*.json` 中，结构：

```json
{
  "name": "my_pipeline",
  "steps": [
    { "tool_id": "echo_sample", "args": ["hello"] }
  ]
}
```

UI 支持：
- 新建 pipeline（名称 + 步骤列表）
- 逐步执行：每执行一步后进入“待确认”状态
- 点击“确认”后才能执行下一步
- 重置执行状态

### 5.6 Knowledge 快速记录

右下角常驻输入框：
- 输入文本后点击“保存”
- 或 Enter（不按 Shift）快速保存

保存位置：workspace 的 `knowledge/entries.jsonl`（每行一条，包含 timestamp/text）

### 5.7 AI Hub

右上角提供外部 AI 页面入口：
- ChatGPT、Claude、DeepSeek
- 千问、元宝、豆包

## 6. 常见问题排查

### 6.1 文件树不显示文件

请确认：
- 后端已启动（`http://127.0.0.1:8000/docs` 可打开）
- workspace 目录中确实有文件（如 `workspace_project/datas/sample.jsonl`）
- 前端通过代理访问后端（在 `frontend/vite.config.ts` 已配置）

### 6.2 工具执行失败

请确认：
- 脚本在 workspace 的 `tools/` 下，且以 `.py` 结尾
- 脚本可被 `python3` 执行
- 脚本运行时间不要超过 300s（当前后端有超时限制）
